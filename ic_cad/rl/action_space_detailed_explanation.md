🎲 Categorical 分布的數學原理:
P(action_i) = exp(logit_i) / Σ exp(logit_j)

當 logit_i = -∞ 時:
exp(-∞) = 0  →  P(action_i) = 0 / Σ = 0

🔒 安全採樣保證:
✅ 無效動作的機率為 0，絕不會被選中
✅ 只有有效動作會被採樣
✅ 動作分布自動重新歸一化到有效空間

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
